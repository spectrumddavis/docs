Returns external item records using the standard [paginated query](/api-reference/common-paginated-query) pattern. External items are a **read-only** aggregate view joining purchase order lines with receipt lines, providing a consolidated view of ordered, received, and invoiced quantities.

**Endpoint:** `GET /api/external-items`

---
