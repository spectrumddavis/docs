Matches an address against your tenant's master data. Returns a ranked list of candidates with confidence scores.

**Endpoint:** `POST /api/addresses/match`

For a conceptual overview of the matching algorithm, shortlisting stages, options (match method, confidence threshold, field weights, fuzzy matching), and result interpretation, see [Matching business entities](/api-guide/matching-business-entities).
