---
title: 'Batch upsert addresses'
openapi: 'api-reference/openapi.json PUT /addresses/batch-upsert'
---

Imports or updates addresses in bulk using the standard [batch upsert pattern](/api-reference/common-batch-upsert).

**Endpoint:** `PUT /api/addresses/batch-upsert`

---

## Fields

| Field | Type | Required | Description |
|---|---|---|---|
| `sourceSystem` | string | Yes | Name of the source system (e.g. `"Sage"`) |
| `sourceSystemId` | string | Yes | Unique identifier in the source system |
| `businessUnitId` | string | Yes | ID of the owning business unit |
| `name` | string | No | Address label or description |
| `addressLine1` | string | No | Address line 1 |
| `addressLine2` | string | No | Address line 2 |
| `postcode` | string | No | Postcode |
| `city` | string | No | City |
| `county` | string | No | County |
| `type` | string | No | Address type (e.g. `Billing`, `Delivery`) |
