---
title: 'Batch upsert suppliers'
openapi: 'api-reference/openapi.json PUT /suppliers/batch-upsert'
---


Imports or updates suppliers in bulk using the standard [batch upsert pattern](/api-reference/common-batch-upsert).

**Endpoint:** `PUT /api/suppliers/batch-upsert`

---

## Fields

| Field | Type | Required | Description |
|---|---|---|---|
| `sourceSystem` | string | Yes | Name of the source system (e.g. `"Sage"`) |
| `sourceSystemId` | string | Yes | Unique identifier in the source system |
| `businessUnitId` | string | Yes | ID of the owning business unit |
| `name` | string | No | Supplier display name |
| `type` | string | No | Supplier classification |
| `companyNumber` | string | No | Companies House number |
| `vatRegistrationNumber` | string | No | VAT registration number |
| `addressLine1` | string | No | Address line 1 |
| `addressLine2` | string | No | Address line 2 |
| `postcode` | string | No | Postcode |
| `city` | string | No | City |
| `county` | string | No | County |
| `bankAccountNumber` | string | No | Bank account number |
| `bankAccountSortCode` | string | No | Sort code |
| `bankAccountIBAN` | string | No | IBAN |
| `bankAccountSWIFT` | string | No | SWIFT/BIC code |
| `telephoneNumber` | string | No | Telephone number |
| `faxNumber` | string | No | Fax number |
| `emailAddress` | string | No | Email address |
| `lineItemsRequired` | boolean | No | Whether line items are required for invoices |
| `purchaseOrdersRequired` | boolean | No | Whether purchase orders are required |
| `useFirstLineItemAsInvoiceDescription` | boolean | No | Use the first line item description as the invoice description |
| `defaultApproverId` | string | No | Default approver user ID |
| `defaultNominalCodeId` | string | No | Default nominal code ID |
| `isActive` | boolean | No | Whether the supplier is active |
